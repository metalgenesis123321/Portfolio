Projects

Facial Ensemble Disentanglement

Developed a facial recognition tool capable of isolating individual identities in group photographs. Implemented using Python with TensorFlow and Keras, the model leverages ensemble learning to disentangle features and accurately identify faces. Deployed on AWS for seamless scalability and reliability.

Medical Image Report Generation

Designed a system that automatically generates radiology reports by combining computer vision and natural language processing. Utilised transformer models and memory‑driven modules to interpret medical images and employed the PubMed BERT model for text generation. Trained and hosted on AWS SageMaker.
